# Vue Sales Funnel

This is a simple Vue project demonstrating how you can quickly build a Sales Funnel using Bootstrap and integrating it with third-party services such as MailerLite.

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Run your tests

```bash
npm run test
```

### Lint and fix files

```bash
npm run lint
```
